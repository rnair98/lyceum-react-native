# lyceum-react-native
a social and educational networking app for finding and saving course recommendations for MOOCs. 

## Installation

1. Clone project

```
git clone https://github.com/rnair98/lyceum-react-native
```

2. cd into folder

```
cd lyceum-react-native
```

3. Download dependencies

```
npm
```

4. Start Expo Server

```
expo start
```
